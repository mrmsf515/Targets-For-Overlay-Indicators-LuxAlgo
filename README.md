# Targets for Overlay Indicators LuxAlgo - Web Application

Professional multi-page web application for **InfinityAlgo Academy**, featuring high-precision trading indicators and documentation.

## Features
- **Professional Design**: Modern, premium light theme optimized for trading tools.
- **Bilingual Support**: Full Arabic (RTL) and English (LTR) support with instant switching.
- **High-Performance**: Built with Vanilla HTML5, CSS3, and JavaScript for maximum speed and compatibility.
- **High-Conversion Checkout**: Integrated "Order Bump" strategy for InfinityRSI Divergence V6.2.
- **Detailed Docs**: Comprehensive documentation and FAQ sections to educate and convert users.
- **Responsive**: pixel-perfect design across all devices (Mobile, Tablet, Desktop).

## Project Structure
- `index.html`: Home page featuring the free indicator.
- `docs.html`: Detailed documentation and institutional-grade FAQ.
- `checkout.html`: Secure checkout with the $12.50 Order Bump offer.
- `styles.css`: Global design system with Cairo font integration.
- `script.js`: Internationalization (i18n) logic and UI interactions.

## Tech Stack
- **HTML5 / CSS3 / ES6+ JavaScript**
- **Fonts**: Cairo (Google Fonts)
- **Theme**: Light Mode / White Minimalist
- **Deployment**: Vercel ready (static deployment)

## Deployment Instructions (Vercel)
1. Push this repository to your GitHub account.
2. Go to [Vercel](https://vercel.com) and click "Add New" -> "Project".
3. Import the `Targets-For-Overlay-Indicators-LuxAlgo` repository.
4. Click "Deploy". Your site will be live instantly!
